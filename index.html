My Skills.

Python

Ich wollte schon immer als Entwickler arbeiten, wusste aber nie, welche Position die richtige für mich ist. Also fing ich an, die beliebteste Programmiersprache für Anfänger zu lernen. Nachdem ich den Online-Kurs 100 Days of Code: the Complete Python Pro Bootcamp 2022 in Udemy von Angela Yu absolviert und viele Projekte durchgeführt hatte (Sie finden sie unten in meinem Github-Konto), fühlte ich mich wohler und begann, mich für Webentwicklung zu interessieren.


HTML

Als ich den Udemy-Kurs beendet hatte, wollte ich mehr über Webentwicklung lernen, und HTML spielt dabei eine große Rolle. HTML ist eine HyperText Markup Language und es definiert die Bedeutung und Struktur von Webinhalten.


CSS

Eine große Rolle jeder Website spielt auch ihr Styling. Cascading Style Sheet, a.k.a. CSS, ist für das Styling und Aussehen einer Website verantwortlich.


JavaScript

Nachdem ich genug über die Struktur der Website und ihr Styling gelernt hatte, bemerkte ich, dass andere Websiten interaktiv sein konnten. JavaScript ist aktiviert Objektorientierte Computerprogrammiersprache, die häufig verwendet wird, um interaktive Effekte in Webbrowsern zu erstellen.


SASS

Sass ist eine Stylesheet-Sprache, die als CSS-Präprozessor mit Variablen, Schleifen und vielen anderen Funktionen, die Cascading Style Sheets nicht beinhaltet, die Erstellung von CSS vereinfacht und die Pflege großer Stylesheets erleichtert.


jQuery

jQuery ist eine freie JavaScript-Bibliothek, die Funktionen zur DOM-Navigation und -Manipulation zur Verfügung stellt. jQuery ist die meistverwendete JavaScript-Bibliothek. 77 % aller Websites und 69 % der 10.000 meistbesuchten Websites nutzen jQuery.
